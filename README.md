Общее описание:
Приложение должно позволять пользователю:
1 создавать свои блюда и указывать их калорийность
2 учитывать съеденные блюда
3 Считать общую калорийность съеденного за день

структура проекта
- папка Model содержит:
    - MenuDish  содержит модель Блюдо меню
    - EatedDish содержит модель Съеденное блюдо
    - Protocols содержит протоколы
- папка View содержит:
    - папка Dishes: содержит view, реализуюшие путь пользователя по созданию редактированию и удалению блюда
        - view AddDish - экран добаввления нового блюда
        - view EditDish - экран редактирования или удаления блюда
        - view DishesListView - экран со списком блюд
    - папка Products содержит view, реализуюшие путь пользователя по созданию редактированию и удалению продуктов. Сейчас не используется в прилдожении
        - view
        - view
    - папка Statistics содержит view, реализуюшие путь пользователя по учету съеденных блюд 
        - view ChooseDish - экран выбора блюда, которое пользователь хочет съесть
        - view EatDish - экран, на котором пользователь вводит количество блюда, которое он хочет съесть
        - view EditEatedDish - экран, на котором пользователь может изменить количество уже съеденного или удалить запись
- ViewController - основной экран приложения со списком съеденных блюд и количеством калорий за день
